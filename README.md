# movietime
The Movie DB client done for testing purposes

## Getting Started

To get a local copy up and running follow these simple steps.

### Installation
 
1. Clone the repo
```sh
git clone https://github.com/aaskrv/movietime.git
```
2. Install the Pods
```sh
pod install
```
3. Open the Project file
```sh
movietime.xcworkspace
```
4. Build and Run
